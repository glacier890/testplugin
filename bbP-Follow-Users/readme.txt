== Description ==

This simple plugin allows users to follow other members on bbPress . The users can view the latest posts and replies from their followed users in a widget that can be placed anywhere on the site.

This plugin adds:


* Follow / Unfollow links that are displayed under each user in the bbPress topics.
* The ability to determine the max number of posts that can be shown in a widget.


For bug reports or to add patches or translation files, visit the [bbPress Follow Users Github page](https://github.com/kcbluewave890/).

== Installation ==

1. Download, install and activate the plugin.
1. To follow a user, simply visit the link under their post in bbPress and hit the follow link under their name.


== Frequently Asked Questions ==

= Can I customize the titles of the widgets? =
Yes. After placing the widgets, you will be able to edit titles for each of the widgets.
